
## Maximising value

In a self-hosted AI setup, a large part of the costs are the up-front, fixed, investments in purchasing computer (in particular GPU and RAM) that can run useful LLM models.
Assuming a low cost of electicity,
the economic incentive would be to use the system as much as possible. 

Which things in software development etc are **valuable** to do **a lot of**?
While still being something that the agents can do with minimal or rare human input? 
Such that it would be useful and meaningful to let agents do it continiously
"as much as possible"?

### Candidates

- Usage examples/demos for software library/frameworks
- Security scans of code
- Active attacts of code/service for security vunerabilities
- Scanning code to find bugs
